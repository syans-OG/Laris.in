# FungsiSkills.md

Dokumen ini berisi daftar kurasi skill Antigravity untuk pengembangan Website Fashion 3D & Frontend.



## 1. Core Agent Intelligence (Wajib Aktif)
*Skill ini adalah "otak" operasional agen. Aktifkan ini di awal sesi agar agen bekerja cerdas dan terstruktur.*

**behavioral-modes**
*   **Fungsi:** Mengubah mode operasi agen (Brainstorm, Implement, Debug, Review).
*   **Pentingnya:** Mencegah agen langsung koding saat Anda hanya ingin diskusi, atau sebaliknya.

**systematic-debugging**
*   **Fungsi:** Menjalankan prosedur perbaikan bug secara sistematis (bukan menebak-nebak).
*   **Pentingnya:** Wajib digunakan saat menghadapi error kode yang sulit agar tidak merusak kode lain.

**loki-mode**
*   **Fungsi:** Mode otonom penuh "Startup-in-a-box" yang mengelola banyak sub-agen.
*   **Pentingnya:** Gunakan hanya jika Anda ingin agen mengerjakan tugas sangat besar secara mandiri (misal: "buatkan seluruh fitur login dari nol sampai selesai").

**file-organizer**
*   **Fungsi:** Mengorganisir file dan folder yang berantakan berdasarkan konteks proyek.
*   **Pentingnya:** Menjaga struktur folder proyek tetap rapi seiring bertambahnya aset 3D.




## 2. 3D & Interactive Specialization (Fitur Utama)
*Skill spesifik untuk membuat website fashion 3D yang interaktif dan menarik.*

**3d-web-experience**
*   **Fungsi:** Spesialis Three.js, React Three Fiber, dan WebGL.
*   **Pentingnya:** Skill paling krusial untuk merender model baju 3D, konfigurator produk, dan scene interaktif.

**scroll-experience**
*   **Fungsi:** Membuat efek *scrollytelling*, paralaks, dan animasi berbasis scroll.
*   **Pentingnya:** Agar presentasi produk terasa sinematik saat pengunjung menggulir halaman (seperti website Apple).

**algorithmic-art**
*   **Fungsi:** Membuat seni generatif dan visual dinamis menggunakan kode (p5.js).
*   **Pentingnya:** Berguna untuk membuat latar belakang (background) artistik yang unik untuk produk fashion.




## 3. Frontend Architecture & Frameworks
*Standar koding untuk memastikan performa dan kemudahan maintenance.*

**frontend-dev-guidelines**
*   **Fungsi:** Panduan utama arsitektur React/TypeScript (struktur folder, lazy loading, state management).
*   **Pentingnya:** Mencegah struktur kode menjadi berantakan ("spaghetti code").

**nextjs-best-practices**
*   **Fungsi:** Menguasai App Router, Server Components, dan data fetching di Next.js.
*   **Pentingnya:** Wajib jika menggunakan Next.js agar aplikasi cepat dan SEO-friendly.

**typescript-expert**
*   **Fungsi:** Ahli dalam tipe data kompleks dan keamanan tipe (*type safety*).
*   **Pentingnya:** Mencegah bug data (misal: properti baju tidak muncul) sebelum kode dijalankan.

**tailwind-patterns**
*   **Fungsi:** Manajemen CSS modern dengan Tailwind v4 dan arsitektur token desain.
*   **Pentingnya:** Menjaga styling tetap konsisten dan mudah diubah.

**react-patterns**
*   **Fungsi:** Pola lanjutan React (Hooks, Composition, Performance).
*   **Pentingnya:** Membantu menulis logika komponen yang efisien.

**react-ui-patterns**
*   **Fungsi:** Menangani pola UI umum (loading states, error boundaries, async data).
*   **Pentingnya:** Memastikan UX tetap mulus meskipun data sedang dimuat.




## 4. UI/UX Design & Visual Strategy
*Skill untuk mendesain tampilan sebelum kode ditulis.*

**ui-ux-pro-max**
*   **Fungsi:** Skill desain visual tingkat tinggi (Glassmorphism, Bento Grid) dan komponen shadcn/ui.
*   **Pentingnya:** Agar tampilan antarmuka website terlihat mahal dan modern.

**frontend-design**
*   **Fungsi:** Fokus pada implementasi kode desain yang "polished" dan detail.
*   **Pentingnya:** Menghindari tampilan kaku khas "template AI".

**web-design-guidelines**
*   **Fungsi:** Auditor desain untuk aksesibilitas dan standar web.
*   **Pentingnya:** Memastikan website nyaman dibaca dan diakses semua orang.

**mobile-design**
*   **Fungsi:** Prinsip desain responsif dan interaksi sentuh (*touch*).
*   **Pentingnya:** Memastikan pengalaman 3D tetap enak digunakan di layar HP.

**theme-factory**
*   **Fungsi:** Pembuat tema dan palet warna otomatis.
*   **Pentingnya:** Membantu menjaga konsistensi warna brand fashion di seluruh halaman.




## 5. Planning & Product Strategy
*Gunakan ini di tahap awal (brainstorming) atau saat merinci fitur baru.*

**brainstorming**
*   **Fungsi:** Mengubah ide abstrak menjadi rencana konkret yang tervalidasi.
*   **Pentingnya:** Gunakan saat Anda bingung "fitur apa lagi yang harus ditambahkan?".

**multi-agent-brainstorming**
*   **Fungsi:** Diskusi desain menggunakan beberapa "persona" agen sekaligus.
*   **Pentingnya:** Mengurangi risiko kesalahan konsep dengan melihat ide dari berbagai sudut pandang.

**product-manager-toolkit**
*   **Fungsi:** Prioritas fitur, dokumen spesifikasi (PRD), dan strategi produk.
*   **Pentingnya:** Membantu Anda fokus pada fitur yang benar-benar penting untuk diselesaikan duluan.




## 6. Optimization, Quality & Deployment
*Tahap akhir untuk memastikan website cepat, bersih, dan siap tayang.*

**web-performance-optimization**
*   **Fungsi:** Audit kecepatan, Core Web Vitals, dan ukuran file bundle.
*   **Pentingnya:** **Sangat Penting** untuk website 3D agar tidak berat saat dibuka.

**clean-code**
*   **Fungsi:** Standar koding pragmatis dan ringkas tanpa komentar sampah.
*   **Pentingnya:** Menjaga kode tetap mudah dibaca manusia.

**lint-and-validate**
*   **Fungsi:** Otomatisasi pengecekan error sintaks dan format kode.
*   **Pentingnya:** Jalankan sebelum commit agar kode yang disimpan selalu bersih.

**vercel-deployment**
*   **Fungsi:** Ahli deploy aplikasi Next.js ke platform Vercel.
*   **Pentingnya:** Membantu proses peluncuran website ke publik.
